# Faceit
基于opencv、dlib与qt的视频、图像人脸互换等功能开发

---

opencv版本：3.4.1

dlib版本：19.7

qt：5.9.0

视频的人脸互换是纯opencv实现的，图像的人脸互换是用opencv+dlib实现的，换起来有点点小问题

除了换脸的功能，还有人脸相似度检测（不是很准）、数字化视频与其他一些opencv基本图像处理
